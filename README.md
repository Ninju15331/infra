## üìñ What is infra?

Infra is a tool designed to help you manage your servers and cloud services with ease. It offers automated configurations and monitoring for various applications and platforms. You don't need to know how to code or work in complex environments to start using it.

This project mirrors the original from GitLab and focuses on making managing technologies like Element, Grafana, Nextcloud, and Synapse simpler. It includes helpful setups for popular tools like Jitsi Meet and Prometheus.
